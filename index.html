Selected Projects

Client Website

Kapgen Tours & Safaris

This website was build for a tours and safaris company named “Kapgen Tours & Safaris”. The purpose is to target international clients and allowing them to book or send message inquiries via the website, making it an easy booking process no matter where the client is from. The website is built with HTML, CSS and JavaScript with Supabase, PostgreSQL for the database which holds a webhook for the automation systems for the customized automatic emails to clients and the company immediately when a new booking or message inquiry is submitted and lands in the database. This website is live and running with future updates still to come as per request from the company.

Live Website
Concept Build

Golden Kalahari Retreat

This website is a concept build for skill development and exploration. The concept is a retreat lodge located in Namibia's Kalahari which serves as the perfect getaway, having a spa included for a relaxing moment. It's a website build just on one page with the following sections: about us, why us, suites, experiences, dining, wellness, gallery, reviews, faq, booking and contact. I made use of Flatpickr to mark unavailable dates on the calender for clients to know which dates are available to book. With this website, I have also made flyers for the retreat and spa.

Portal System

University Student Portal

This smart campus portal was build as a project aiming for skill improvement and database testing. It allows admins or students to sign up and have access to their aligned responsiblities. Though there's an assigned admin that is secured, any student can sign up and will create their own passwords. The admin can post/edit events or announcements, view feedback and complaints from students, act on pending course registrations from students and add more courses. The students are able to log in and view the announcements, register to new courses, see the register course status whether it's accepted, pending or declined and submit feedback or complaints.
